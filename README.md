# log-archive-tool

CLI tool to archive logs on a set schedule by comrepssing them and storing them in a new directory

# How to use it?

1. Clone the repository or copy the content of (Log-Archive)[log-archive.sh]

```
git clone git@github.com:reneb1404/log-archive-tool.git
cd log-archive-tool
```

2. Make the script executable

```
chmod +x log-archive.sh
```

3. Execute the script

```
./log-archive.sh {PathToFolderToArchive}
```
